## 📁 Struktur Proyek

```
marcel-karamoy-portfolio/
│
├── index.html             → Halaman utama website
├── style.css              → Styling dasar menggunakan CSS
├── mediaqueries.css       → Responsiveness via media queries
├── script.js              → Interaksi dasar (hamburger menu, dll.)
```
